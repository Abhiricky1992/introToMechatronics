Introduction to Control

Open-Loop Control

  • The plant output doesn't affect the control signal.
  • Very easy and cheap to implement.
  • Reliability depends on the quality of calibration.
  • Cannot handle external disturbances.

Closed-Loop Control

4.9
  • The output of the plant is fed back to calculate the error.
  • Difficult to implement since the control signal is a function of the error.
  • Reliability is generally high since minimal calibration is required.
  • Can tolerate external disturbances.

Terminology

4.9
  • Output \((x)\): A quantity or parameter of the plant that changes because of the changing control signal.
  • Set-point \((x_{des})\): An output value that the plant should reach. Other names are 'Reference Input' or 'Desired State'.
  • Error \((e)\): The difference between the set-point and the plant output at any given time.
  • Control Signal \((\tau)\): The quantity or the parameter of the plant that can be adjusted by a controller for achieving zero error. The control signal for the sliding block example would be the throttle.

Cautions

  • There are always limits to what a sensor can measure accurately. The controller should not react to such less accurate measurements.
  • Many sensors cannot provide measurements continuously. This discreteness in sensor measurement can affect the performance of a PID controller.
  • Allowing the integral control values to change without bounds can cause an overflow/underflow in the variable that stores the integration value.
  • The control signal generally has a maximum and a minimum value. Thus, it is important to restrict the value of the control signal.
  • If a positive control signal value reduces the error then the control gains must be positive, otherwise the control gains must be negative.
