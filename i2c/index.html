I2C Communication Protocol

I2C Communication Protocol

  • I2C stands for Inter-Integrated Circuit.
  • It is a two wire communication system, one for data signal and one for clock signal.
  • Multiple devices can be connected on a set of wires.
  • Communication is controlled by one device only.
  • This is a synchronous communication protocol since all devices operate on the same clock signal.
  • Following are the baud rates available for I2C protocol.
    Mode Standard Fast Fast+ High-Speed Ultra-Fast
    Speed 100 kb/s 400 kb/s 1 Mb/s 3.4 Mb/s 5 Mb/s
