Information Storage in MCU

FLASH

  • Non-volatile
  • Holds the complete code and data
  • Large storage capacity (MBs to GBs), low speed

RAM

  • Volatile
  • Holds the code and data that is in the process
  • Moderate storage capacity (KBs to MBs), higher speed than FLASH

Register

  • Volatile, may have a reset value
  • Holds the data that CPU is working on right now
  • 16 or 32 bits per register, fastest data access
